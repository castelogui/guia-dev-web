Guia HTML para Iniciantes

Aprenda o básico sobre como criar páginas da web usando HTML!

1. O que é HTML?

HTML significa HyperText Markup Language, ou seja, Linguagem de Marcação de Hipertexto. Ele é usado para estruturar o conteúdo da web.

2. Para que serve?

  • Exibir textos, imagens e vídeos no navegador
  • Organizar conteúdo como parágrafos, links, títulos e listas
  • É a base de qualquer página web

3. Como criar seu primeiro arquivo HTML?

  1. Abra um editor de texto como o Visual Studio Code
  2. Crie um novo arquivo e salve como index.html
  3. Copie e cole a estrutura abaixo

4. Estrutura básica de um HTML

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Minha Primeira Página</title>
  </head>
  <body>
    <h1>Olá, mundo!</h1>
    <p>Essa é a minha primeira página web.</p>
  </body>
</html>

5. Principais Elementos

Tag Descrição
<h1> a <h6>Títulos (do maior ao menor)
<p>Parágrafos de texto
<a href="">Links para outros sites
<img src="">Imagens
<ul>, <ol>, <li>Listas
<div>Bloco de conteúdo
<span>Trecho de texto

6. Como visualizar no navegador?

  1. Salve o arquivo como index.html
  2. Dê dois cliques nele
  3. Ele será aberto no seu navegador padrão

7. Prática simples: Página pessoal

Copie e cole este código no seu arquivo index.html:

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Sobre mim</title>
  </head>
  <body>
    <h1>Olá! Eu sou a Maria</h1>
    <p>Estou aprendendo desenvolvimento web.</p>
    <h2>Meus hobbies</h2>
    <ul>
      <li>Ler livros</li>
      <li>Ouvir música</li>
      <li>Aprender coisas novas</li>
    </ul>
    <p>Visite meu <a href="https://www.exemplo.com">site favorito</a>.</p>
  </body>
</html>

8. Próximos passos

  • CSS – para estilizar (cores, fontes)
  • JavaScript – para interações
  • Responsividade – para celulares
  • Hospedagem – para colocar seu site no ar
