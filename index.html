Guia Completo de HTML - Aprenda em 1 Mês

Este guia foi pensado para te ensinar HTML do zero ao essencial em 4 semanas, com teoria, prática e desafios semanais.

Semana 1: Fundamentos do HTML

  • O que é HTML?
  • Tags básicas: <html>, <head>, <body>, <h1> até <h6>, <p>
  • Listas: <ul>, <ol>, <li>
  • Links e imagens: <a href>, <img src>
  • Prática: Criar uma página "Sobre mim" com seus hobbies, uma foto e um link

Semana 2: Organização de Conteúdo

  • Estrutura de seções com <div> e <section>
  • Uso de <span>, <br> e <hr>
  • Criação de tabelas com <table>, <tr>, <td> e <th>
  • Formatação de texto: <strong>, <em>, <mark>, <small>
  • Prática: Criar uma tabela de tarefas e uma ficha de personagem

Semana 3: Formulários e Acessibilidade

  • Formulários: <form>, <input>, <textarea>, <label>, <select>, <option>
  • Atributos importantes: placeholder, value, required, type
  • Boas práticas de acessibilidade: uso de <label for>, estrutura semântica
  • Prática: Criar um formulário de cadastro com campos obrigatórios

Semana 4: Projeto Final e Otimizações

  • Revisar tudo aprendido
  • Organizar HTML com comentários
  • Aplicar layout com <header>, <nav>, <main>, <footer>
  • Inserir ícones e vídeos (usando <iframe> ou <video>)
  • Projeto final: Criar uma página com currículo, formulário de contato e portfólio de links

Extras Úteis

  • Validação do código: https://validator.w3.org
  • Ícones gratuitos: https://fontawesome.com ou https://icons.getbootstrap.com
  • Imagens gratuitas: https://unsplash.com
  • Hospedagem grátis: https://vercel.com ou https://netlify.com
