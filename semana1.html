Semana 1: Fundamentos do HTML

Vamos começar com os blocos fundamentais para montar qualquer página da web.

1. O que é HTML?

HTML significa HyperText Markup Language e é a linguagem usada para estruturar o conteúdo de páginas web. Ele não é uma linguagem de programação, mas sim de marcação.

2. Tags Básicas

Essas são as primeiras tags que você precisa conhecer:

  • <html> - Indica o início do documento HTML
  • <head> - Contém informações sobre o documento (como título e codificação)
  • <body> - Onde vai o conteúdo visível
  • <h1> até <h6> - Títulos de níveis diferentes
  • <p> - Parágrafos de texto

Exemplo:

<!DOCTYPE html>
<html>
  <head>
    <title>Minha Página</title>
    <meta charset="UTF-8">
  </head>
  <body>
    <h1>Título Principal</h1>
    <p>Este é um parágrafo de exemplo.</p>
  </body>
</html>

3. Listas

HTML tem dois tipos principais de listas:

  • <ul> - Lista não ordenada (com marcadores)
  • <ol> - Lista ordenada (com números)
  • <li> - Cada item da lista

Exemplo:

<h2>Lista de compras</h2>
<ul>
  <li>Pão</li>
  <li>Leite</li>
  <li>Café</li>
</ul>

<h2>Passos para fazer café</h2>
<ol>
  <li>Ferver a água</li>
  <li>Adicionar pó de café</li>
  <li>Coar e servir</li>
</ol>

4. Links e Imagens

  • <a href="URL">Texto</a> - Cria um link
  • <img src="caminho" alt="descrição" /> - Exibe uma imagem

Exemplo:

<p>Visite o <a href="https://www.google.com" target="_blank">Google</a>.</p>
<img src="https://via.placeholder.com/150" alt="Imagem de exemplo" />

5. Prática da Semana: Página "Sobre mim"

Crie um arquivo index.html com o seguinte conteúdo:

<!DOCTYPE html>
<html>
  <head>
    <title>Sobre mim</title>
    <meta charset="UTF-8">
  </head>
  <body>
    <h1>Olá! Eu sou a Maria</h1>
    <p>Estou aprendendo HTML nesta semana.</p>

    <h2>Meus hobbies</h2>
    <ul>
      <li>Ler livros</li>
      <li>Ouvir música</li>
      <li>Aprender HTML</li>
    </ul>

    <img src="https://via.placeholder.com/200" alt="Minha foto" />

    <p>Veja o site que eu gosto: <a href="https://www.exemplo.com" target="_blank">Exemplo</a></p>
  </body>
</html>
